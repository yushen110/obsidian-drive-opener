# Drive Opener

把文件资源管理器搬进 Obsidian。

## 功能

- 💾 **盘符浏览**：图形化选择 C/D/E 盘，像资源管理器一样逐层翻
- 🗜️ **压缩/解压**：文件夹右键 → ZIP / 7z / RAR，支持 7-Zip
- 👁️ **文件预览**：图片、PDF、文本直接在 Obsidian 里看
- 🏢 **WPS 集成**：Word/Excel 转 PDF、PDF 合并拆分、批量打印、格式互转
- ✏️ **批量重命名**：序号前缀、去空格、转小写、正则替换，预览后执行
- 📋 **多点剪贴板**：记多批复制/剪切，统一粘贴
- 🔗 **插入引用**：一键插入文件链接或文件夹索引到笔记
- 🖱️ **拖拽支持**：拖文件到文件夹自动移入，拖出 Obsidian 到外部

## 安装

### 手动安装

1. 下载 `main.js` 和 `manifest.json`
2. 放入 vault 的 `.obsidian/plugins/drive-opener/` 目录
3. 在 Obsidian 设置 → 社区插件中启用

### 前置条件（可选）

- **7-Zip**：安装后可压缩为 7z/RAR 格式，ZIP 兼容性更好
- **WPS Office**：安装后可使用文档转换、PDF 合并拆分、批量打印

## 系统要求

- Obsidian >= 0.15.0
- Windows（目前仅支持 Windows）
