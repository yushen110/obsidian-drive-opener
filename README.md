# Drive Opener

把文件资源管理器做成了 Obsidian 插件。

## 能做什么

在 Obsidian 里直接浏览电脑所有盘符和文件，不用切到资源管理器。插件打开后是一个盘符选择界面，C 盘 D 盘点一下就能进，跟资源管理器一样逐层往下翻。文件夹和文件支持右键操作、拖拽、多选。

## 功能

**文件浏览**
- 图形化盘符选择，也支持手动输入路径
- 文件列表支持搜索过滤
- 图片、PDF、文本文件可直接预览

**压缩/解压**
- 文件夹右键直接压缩为 ZIP / 7z / RAR
- 支持解压 ZIP / 7z / RAR / TAR / GZ / BZ2 等格式
- 优先调用 7-Zip（更高的兼容性和压缩率），未安装时回退到 PowerShell

**WPS Office 集成**
- Word / Excel / PPT 一键转 PDF
- 多个 PDF 合并为一个
- PDF 按页拆分
- 批量打印
- 格式互转（DOCX ↔ PDF ↔ TXT）

**批量重命名**
- 查找替换（支持正则）
- 预设规则：序号前缀、去空格、转小写、日期前缀
- 预览确认后再执行

**多点剪贴板**
- 可以记多批复制/剪切操作，选择粘贴
- 同时写入系统剪贴板，粘贴到外部资源管理器也有效

**其他**
- 新建文件夹 / 文本文档 / Word / Excel
- 插入文件链接或文件夹索引到当前笔记
- 拖拽文件到文件夹自动移入
- 拖出 Obsidian 到外部窗口（文件路径）
- 移动到回收站

## 安装

下载 `main.js` 和 `manifest.json`，放入 vault 的 `.obsidian/plugins/drive-opener/` 目录，在 Obsidian 设置中启用。

## 前置条件（可选）

- **7-Zip**：安装后可压缩为 7z/RAR，ZIP 兼容性更好
- **WPS Office**：安装后可使用文档转换、PDF 合并拆分、批量打印

## 系统要求

- Obsidian ≥ 0.15.0
- Windows（当前仅支持 Windows）
